# Keats' Home v15.0 Alpha｜主屋重制

这一版第一次把两项已经确定的核心合在一起：

1. **Keats 正式视觉建模**
2. **Keats Autonomous Core v1.1 自主判断**

## 当前实现

- 正式 Keats 形象进入主屋，不再继续使用旧版程序小人
- 清醒猫人态、坐姿与睡眠猫态可切换
- “叫 Keats、摸摸、坐到沙发、添茶、说我需要你”等事件先进入决策层
- Keats 会根据独立性、偏爱、精力、独处需求、当前行为与行为惯性判断
- 面板可展开查看本次选择的原因与候选意图得分
- 无操作时，Keats 会自主看书、看窗、坐着或睡觉
- 手机与电脑响应式布局
- 浏览器本地保存状态

## 打开方式

打开本目录的 `index.html`。

## Alpha 限制

- GitHub 版本直接调用 `visual-bible/previews/` 中的正式视觉预览，后续会替换为真正透明分层角色素材
- 主屋家具目前是统一绘本风格的布局原型，后续继续资产化
- 自主核心仍处于人格校准阶段，尚未标记为最终 `LOCKED`
