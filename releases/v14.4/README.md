# Keats' Home v14.4｜主房间双端稳定样板

这是新一代小屋的主房间交互样板，用于验证清晰场景、手机/电脑触控、现实时间、灯光与窗帘、地面行走、纸条微剧情和 Keats 自主行为。

## 最方便的打开方式

直接下载：

- `Keats_Home_v14.4_standalone.html`

这是 GitHub Actions 自动合成的完整单文件 HTML，下载后直接打开即可。

## 可维护源文件

- `Keats_Home_v14.4_main_room.html`：页面结构
- `style.css`：场景与双端样式
- `app.js`：生活、剧情、存档与交互逻辑

源文件应保持在同一目录；修改后，工作流会重新生成 standalone HTML。

## 与 v13 的关系

- v13：目前完整、稳定的四房间小屋，入口为仓库根目录 `index.html`
- v14.4：新画面与新交互架构的主房间样板，暂不替代 v13

## 已验证重点

- 手机端与电脑端使用同一套逻辑
- 点击地面时 Keats 平滑行走，不瞬移
- 摸摸不会与地面移动串台
- 窗帘有可见的开合变化
- 纸条可点击，重复阅读不会重复增加亲密度
- 睡眠交互不会偷偷改变醒睡状态
- 生活、剧情和设置分开保存
