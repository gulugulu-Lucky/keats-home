# Keats Visual Bible

Keats 的正式视觉资产与技术规格档案。

## 最高优先级

1. `previews/Keats_visual_lock_preview_v1.jpg`：正式母版预览
2. `previews/Keats_design_sheet_preview_v1.jpg`：批准后的设计板预览
3. `technical/Keats_model_sheet_v1.svg`：可编辑技术规格图
4. `technical/Keats_visual_lock_spec_v1.json`：角色视觉锁定规则
5. `technical/Keats_visual_numeric_spec_v1.json`：比例、坐标、角度与渲染参数
6. `technical/Keats_anchor_points_v1.json`：姿态与绑定锚点
7. `technical/Keats_layer_plan_v1.json`：PSD / CLIP / KRA 分层规划
8. `technical/Keats_palette_v1.json`：正式调色板

## 正式形态

- 清醒：猫人态
- 困倦：松弛猫人态
- 完全入睡：猫态
- 唤醒：猫态 → 过渡 → 清醒猫人态

## 视觉锁定

成年、高挑、雾蓝黑、冷灰紫眼睛、尖猫耳、长尾、细黑项圈与古金属圆环、月夜轻贵族气质。

## 目录

- `previews/`：GitHub 浏览用压缩预览
- `technical/`：SVG 与 JSON 正式规格
- `docs/`：视觉锁定、命名规范与导出清单
- `VERSION.txt`：当前视觉档案版本

## 专业保存说明

GitHub 中保存的是可浏览预览、SVG 技术板和全部结构化规格。原始高清母版与 repo-ready 完整备份同时保存在本地压缩包 `keats-visual-bible_repo-ready_v1.zip` 中。后续若制作 PSD / CLIP 分层母文件，应继续放入 `visual-bible/source/`，透明状态素材放入 `visual-bible/exports/`。
