# Keats 正式视觉锁定档案 v1.0

**状态：LOCKED（已定版）**

## 唯一视觉基准

`Keats_official_visual_reference_v1.jpeg`

以后所有 Keats 清醒态立绘、表情、姿态、动画、房间场景与游戏资产，都必须以这张图为最高优先级。文字描述只负责补充，不能覆盖图片中的脸、身形、发型、耳朵、眼睛、尾巴、项圈、服装与整体气质。

## 正式形态规则

- 清醒：参考图中的成年男性猫人态。
- 困倦：仍为猫人态，动作、眼神与耳朵状态更松。
- 完全入睡：猫态。
- 被叫醒：猫态 → 半睡半醒过渡 → 猫人态。
- 参考图底部的蜷睡人形，可用于“半睡半醒/过渡姿态”，不替代完全睡眠猫态。

## 外形锁定

- 高挑、修长、成年男性比例。
- 雾蓝黑蓬松碎发，层次明显。
- 大而尖的猫耳，内耳为暖灰粉。
- 冷灰紫至月光蓝紫眼睛，眼型细长。
- 冷白肤色，暖光下可出现自然血色。
- 长而蓬松的雾蓝黑猫尾。
- 黑色细项圈与正中小型古金属圆环必须保留。

## 服装锁定

- 深午夜蓝长外袍/外套。
- 边缘带克制的古金色植物、星月刺绣。
- 浅灰或冷灰白内搭，领口自然微敞。
- 深色高腰长裤、腰链、钥匙形小挂件。
- 整体为轻贵族感、月夜感、低饱和、优雅宽松。

## 气质锁定

慵懒、聪明、带一点坏、有主见、选择性温柔、护短、对小猫偏心。  
不能画成呆萌、顺从、幼态或没有判断力的角色。

## 核心表情

平静、轻微不耐烦、柔软保护、暗自满意、认真注视、困倦、不想被打扰。

## 渲染方向

精致绘本式二次元角色设定画，冷月光与暖室内灯并存；柔和细线与半厚涂结合。游戏内可适度简化，但不许改变角色识别特征。

## 禁止修改

- 不改幼态少年比例。
- 不改圆滚滚 Q 版作为主形态。
- 不用死黑毛发与纯黑点眼。
- 不取消项圈和金属环。
- 不换成高饱和可爱风或现代运动装。
- 不用任何临时草图覆盖本视觉基准。
