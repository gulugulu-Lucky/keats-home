# Naming Convention

视觉档案目录：`visual-bible`

## 文件命名
- `Keats_visual_lock_master_v1.jpeg`
- `Keats_design_sheet_v1.png`
- `Keats_model_sheet_v1.svg`
- `Keats_visual_numeric_spec_v1.json`
- `Keats_palette_v1.json`
- `Keats_layer_plan_v1.json`
- `Keats_anchor_points_v1.json`

## 资产命名模板

`Keats_[state]_[view]_[variant]_v1.ext`

例如：
- `Keats_awake_front_base_v1.png`
- `Keats_sleep_cat_side_v1.png`
- `Keats_expression_calm_v1.png`
